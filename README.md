# Angular-study
Angular study

## 개발 환경

### 1. node 설치
<a href="https://nodejs.org">https://nodejs.org</a>

### 2. angular cli 설치
npm i @angular/cli -g

### 3. 기본 프로젝트 생성
ng new [프로젝트명]

### 4. 서버 시작
ng serve

### 5. 테스트페이지 확인
<a href="http://localhost:4200">http://localhost:4200</a>

## Web Ui 프레임워크 DevExtreme 사용
프레임워크이지만 커스텀을 할 수 있는 방법이 매우 다양하게 구현되어 있어서 마음대로 커스텀 가능.
